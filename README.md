# llms-in-chess
♟️ LLM Chess Explanations Benchmark

This repository contains the code and experiments for evaluating how Large Language Models (LLMs) explain chess positions in natural language.

The project focuses on comparing multiple LLMs in terms of:

Correctness of the suggested moves and variations
Quality of reasoning behind the explanations
Clarity and instructional value for human learners
Consistency in interpreting board states

We introduce a structured evaluation framework that separates:

Structural understanding (piece placement, legality, board geometry)
Reasoning quality (tactical motifs, plans, and justification)

The benchmark includes:

A curated dataset of chess positions
Prompting strategies for generating explanations
Evaluation metrics and scoring methodology
Scripts for running experiments and aggregating results

This work is part of ongoing research on improving AI-assisted chess learning, with a focus on making model explanations more reliable, educational, and human-like.
